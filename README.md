# cop4331-contact-manager

## Project Description

This is a full-stack contact manager web app for COP 4331.  Users will be able to register, log in, and manage contacts using create, search, update and delete functions. 

## Team Members and Roles

- Gia Do - API
- Tahje Jackson - Database
- Andres Palau - Project Manager, Front End, Float
- Bao Tran - Front End

- ## Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: PHP
- Database: MySQL
- Stack: Lamp
- API Documentation: Swaggerhub

# Features 

- User registration
- User login
- Add contact
- Search contacts
- Update contact
- Delete contact
- Remote server deployment through digital ocean
- SwaggerHub API documentation
- ERD
- Use Case Diagram
- Gantt Chart
- Lighthouse test
